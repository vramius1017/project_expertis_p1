Application PEAPP :

Cette application est formé d'un index lié à 4 pages représentant les quatres fonctionalités ( creation sapin, polish calculator , words spliter et words).

Chaque fonctionalité est structuré par un page contnant un formulaire avec l' url / routé vers une page d'affichage du resultat d'url /result_.

Chaque page / appelle un script js result_.js utilisant la fonction ajax pour serialiser un JSON . Ce JSON contient les données récupérés dans le formulaire de /.

Chaque page / comprend un heaader , un footer et un bouton retour à l'index ( la page d'acceuil de l'application). Chaque page /result_ comprend un heaader , un footer , un bouton retour à l'index de l'application et un bouton de reinitialisation de la fonction.

De plus , dans le fichier views.py , les fonctions de routage des pages / cree un autre json comprenant , en plus des données issue du formulaires , les différentes variables utilisé pour obtenir le resultat. Ces derniéres sont passé dans la vue correspondaante à /result_ par la fonction render_template().
